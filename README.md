# 💣 Bomberman NES Clone

Um clone fiel do clássico Bomberman do NES, desenvolvido do zero com fins de aprendizado e prática de programação.

## 🎮 Sobre o Projeto

Este projeto é uma recriação do Bomberman original lançado para o Nintendo Entertainment System (NES). O objetivo principal é aprofundar conhecimentos em desenvolvimento de jogos, lógica de programação, e estruturas de dados através da implementação prática de um jogo clássico.

## ✨ Funcionalidades

- 🗺️ Mapa em grid com blocos destrutíveis e indestrutíveis
- 💥 Sistema de explosão em cruz com propagação e colisão
- 🏃 Movimentação do jogador com detecção de colisão
- 👾 Inimigos com IA básica
- ⏱️ Sistema de timer por fase
- 🎯 Pontuação e vidas

## 🛠️ Tecnologias Utilizadas

- Linguagem: Python
- Engine/Framework: Pygame

## 🚀 Como Executar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/bomberman-clone.git

# Acesse a pasta do projeto
cd bomberman-clone

# Instale as dependências
npm install  # ou pip install -r requirements.txt

# Execute o jogo
npm start  # ou python main.py
```

## 🎯 Controles

| Tecla       | Ação            |
|-------------|-----------------|
| `↑ ↓ ← →`  | Mover jogador   |
| `Espaço`    | Soltar bomba    |
| `ESC`       | Pausar          |
